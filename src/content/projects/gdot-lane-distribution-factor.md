This project aimed to evaluate the variation of what is called the Lane Distribution Factor (LDF), which is an important traffic feature that is utilized in pavement design. The goal of the project was to update the design LDF values that GDOT implements in their pavement design manual. By using traffic data from GDOT's statewide traffic collection program, the LDF could be estimated. A couple of methods were employed in this project: a hierarchical logistic regression model and an MLP model. An interactive app was created to visualize the effect the differing variables have on the LDF.
