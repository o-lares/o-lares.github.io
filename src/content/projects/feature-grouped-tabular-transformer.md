This study aims to combine data from a variety of sources including crash report data, Continuous Count Station (CCS) data, weather data and pavement data to form a highly descriptive dataset for crash modeling. Several machine learning methods were explored and compared with a novel F-GTT model that was introduced which utilizes a Transformer encoder architecture for enhanced predictive power. Our proposed model uses various features grouped into distinct feature group tokens to allow for enhanced semantic understanding of crash instances to better predict crash types throughout the state of Georgia. Our proposed model outperformed the other machine learning methods that were explored while still allowing for interpretability of results.
