In this project I designed and productionized a GenAI-powered tool using AWS Bedrock, AWS Strands Agents SDK, and Claude Haiku 4.5 to automate large-scale website audits.

The Dealer Dot Com (DDC) support teams previously relied on manual audits of automotive dealership websites to evaluate content quality and compliance, which was time-consuming and inefficient. I developed an agent-based workflow that utilizes LLMs to intelligently analyze and evaluate web content at scale, significantly reducing the manual effort required for audits. The tool was successfully deployed into Posit production as a Streamlit app, generating structured outputs for audit reports that are then presented as client-facing deliverables.
